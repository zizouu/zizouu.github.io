### 변수형
- var : scope가 method 단위 일반적으로 생각하는 code block 단위가 아니다 그 외 선언전에 변수를 사용해도 사용됨.. (사용하지마라)
- const : code block 단위 scope를 갖는다. primitive type 에서는 상수처럼 사용. 참조형 변수는 const 를 사용
- let : primitive type 에서 일반 변수
