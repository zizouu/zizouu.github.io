## django setting
* 가상환경에서 python 명령어를 통해 django프로젝트의 모듈을 사용하려면 환경변수를 잡아줘야한다.
```

set DJANGO_SETTINGS_MODULE=<app-name>.settings  // 윈도 cmd
export DJANGO_SETTINGS_MODULE=<app-name>.settings  // 리눅스 shell

```
